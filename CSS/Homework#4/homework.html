value ใน display property มีอะไรบ้าง
inline แสดงองค์ประกอบเป็นองค์ประกอบแบบอินไลน์
block แสดงองค์ประกอบเป็นองค์ประกอบ Block โดยเริ่มบรรทัดใหม่เสมอ
contents ทำให้ Container หายไป และทำให้ทำให้องค์ประกอบย่อยเป็นประกอบย่อยของถัดไป
flex สร้าง block-level flex container
grid สร้าง block-level grid container
inline-block เป็นองค์ประกอบของ inline-level block container โดยจัดรูปแบบเป็น inline แต่ยังสามารถเพิ่มลด ความยาว สูง ได้
inline-flex เป็นองค์ประกอบของ inline-level flex container
inline-grid เป็นองค์ประกอบของ inline-level grid container
inline-table เป็นองค์ประกอบของ inline-level table
list-item มีลักษณะคล้ายกับ li
run-in เป็นองค์ประกอบของ block or inline, context
table มีลักษณะคล้ายกับ table
table-caption มีลักษณะคล้ายกับ caption
table-column-group มีลักษณะคล้ายกับ colgroup
table-header-group มีลักษณะคล้ายกับ thead
table-footer-group มีลักษณะคล้ายกับ tfoot
table-row-group มีลักษณะคล้ายกับ tbody
table-cell มีลักษณะคล้ายกับ td
table-column มีลักษณะคล้ายกับ col
table-row มีลักษณะคล้ายกับ tr
none องค์ประกอบจะถูกลบออกอย่างสมบูรณ์
initial ตั้งค่าคุณสมบัตินี้เป็นค่าเริ่มต้น
inherit สืบทอดคณสมบัติจาก องค์ประกอบตัวลูก


grid และ flex แตกต่างกันอย่างไร
flex เมื่อ layout มีลักษณะการวางของต่างๆเป็นแถวเดียว ไม่แนวตั้งก็นอน (1D) grid เมื่อ items ด้านในเรามีการจัดเรียงตัวที่ไม่ได้มีแค่แถวเดียว แต่มีลักษณะเป็น column กับ row (2D)


justify and align แตกต่างกันอย่างไร
justify : ใช้จัดเรียงวัตถุในแนวนอนtd> align : ใช้จัดเรียงวัตถุในแนวตั้ง
position คืออะไร และ มีอะไรบ้าง
คือการกำหนดจุดพิกัดในการแสดงผล
  1. static
  2. relative
  3. absolute
  4. fixed


z-index คืออะไรใช้อย่างไร
ใช้จัดเรียง element ที่ซ้อนกันและมีผลกับ element ที่กำหนด position ไว้เท่านั้นครับ (เช่น position:absolute; ,position:relative;)
