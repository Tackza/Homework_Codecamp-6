CSS syntax ประกอบด้วยอะไรบ้าง
Selector ส่วนไหนใน html ที่เราต้องการแต่ง
Property คุณสมบัติในการจัดรูปแบบการแสดงผล
Value ค่าที่เราจะกำหนดให้กับ property


การเขียน CSS มีกี่แบบอะไรบ้าง
1.Inline การเขียนแทรกลงไปในบรรทัดนั้นๆ ของ tag
2.Internal(embedded) เขียนแยก CSS ออกจาก tag แต่อยู่ในหน้าเดียวกันกับไฟล์ HTML
3.External การเขียน แยก css ออกมาเป็นไฟล์ .css ต่างหาก แล้วนำเข้าไปใช้ในหน้าเว็บ


selector มีกี่แบบอะไรบ้าง
1.Class Selectors เป็น Selector ที่เราต้องกำหนดชื่อของ Selector ขึ้นมาเอง โดย Class Selector การเรียกใช้งานต้องระบุชื่อ Class Selectors ให้กับ Tag ของ HTML จึงจะสามารถใช้งานคุณสมบัติของ Class ที่ได้สร้างไว้ได้
2.ID Selectors เป็น Selector ที่อ้างอิงกับ Attribute ที่ชื่อว่า ID ซึ่งเขียนอยู่ภายใน Tag ของ HTML โดยที่ Tag ของ HTML ใดมี ID ซึ่งตรงกับ ID Selectors ที่กำหนด คุณสมบัติของ CSS ที่กำหนดไว้กับ Selector ก็จะมีผลต่อ Tag นั้น
3.TAG Selectors เป็น Selector ที่อ้างอิงกับชื่อ Tag ของ HTML โดยตรง


combinator มีกี่แบบอะไรบ้าง
1.Descendant combinators การเขียน Selector เพื่อสั่งการทำงานทั้งหมด ทั้ง ลูก หลาน โดยการเขียนนั้น ใช้ "ช่องว่าง" ในการสั่งงาน เช่น article li { } หมายถึง li ทั้งหมดที่มีอยู่ใน article
2.Child combinators การสั่งงานเฉพาะ "ลูก" เท่านั้น ไม่นับรวมตัวอื่นการเขียนนั้น ใช้ "เครื่องหมายมากกว่า หรือ greater-than sign (>)" เช่น section>div { } คือ ให้กระทำการใดๆ กับ div ที่เป็น ลูกของ section เท่านั้น ถ้ามี div ซ้อนอยู่ใน div อีกที จะไม่ถูกสั่งการ
3.Sibling combinators Adjacent sibling combinator คือ "น้องที่อยู่ถัดไป 1 คนเท่านั้น" ไม่นับคนที่ 3 หรือ คนอื่นๆ การเขียนนั้น ใช้ "เครื่องหมายบวก (+)" เช่น section+section { } คือ ให้กระทำการใดๆ กับ section ที่อยู่ต่อจาก section ที่เป็นพี่เท่านั้น
General sibling combinator คือ "น้องที่อยู่ถัดไปทุก ๆ คน" ใช้ "เครื่องหมายตัวหนอน หรือ tilde (~)" เช่น h1~p { } คือ ให้กระทำการใดๆ กับ p ที่อยู่หลัง h1 ไม่ว่าจะอยู่ในลำดับที่เท่าไร


property และ value คืออะไรแตกต่างกันอย่างไร
Property คือชื่อคุณสมบัติของ css เช่น width, height, position ฯลฯ
Values คือค่าของ property นั้นๆ ครับ เช่น 500px สำหรับความกว้าง


property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง
1.Property of content
2.Property of border
3.Property of padding
4.Property of margin
5.Property of element


value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร
1.Absolute lengths คือ กำหนดแบบอัตราส่วน เช่น px (pixels, สัมพันธ์กับค่าความละเอียดของหน้าจอ) เช่น 1px, 4px
2.Relative lengths กำหนดตายตัว in (inches; 1in=2.54cm =72pt =6pc) เช่น 2in, 1.5in
